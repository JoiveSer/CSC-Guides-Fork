# CSC Field Guide — CSC 3.3 V5

Статический справочник по Custom Steve Chaos:

- 61 персонаж;
- отдельные СОЛО-закупы PvP и PvE, советы, контры и анализ V5;
- тир-лист персонажей с оценкой, ролью, сильными и слабыми сторонами;
- сохранённые закупы Лингдиана и s1aiRaduga;
- интерактивный подборщик ДУО прямо на главной;
- 1891 сочетание V5, включая зеркальные пики;
- абсолютная редакционная оценка пары, confidence, роли, PvP/PvE-закупы и контр-корректировки;
- адаптивный интерфейс с мобильным бургер-меню;
- полностью локальные данные и иконки;
- публикация на GitHub Pages через `PUBLISH_TO_GITHUB.bat`.

Оценки V5 являются теоретическими и не считаются серверными винрейтами или пикрейтами.

## Локальный запуск

Запустите `START_SITE.bat`.

## Публикация

Запустите `PUBLISH_TO_GITHUB.bat`. Скрипт перезапишет ветку `main` репозитория `JoiveSer/CSC-Guides-Fork` чистой версией сайта.
